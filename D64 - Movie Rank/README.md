# Movie Rank
In this project, I will be creating a Movie Ranking website utilizing
- Flask
- WTForms
- SQLite
- SQLAlchemy
and more

## Requirements
- [X] Be able to view movie list items 
- [X] Be able to edit a movie's rating
- [X] Be able to delete movies from the database
- [X] Be able to add new movies via the Add page
- [X] Be able to sort and rank movies by rating

## Remarks
This was challenging. I found it particularly challenging reading the AlchemySQL documentation. I believe that to be a user skill issue though.

- The Quick Start for Flask-SQLAlchemy was easier to read and ended up primarily being the resource I would use although limited (of course because it's supposed to just walk me through basic use).
- The first hint for the last requirement gave it all away. Sometimes when looking at the requirements or instructions it also felt very limited and vague. But maybe that is a good thing because I'm supposed to Google and read documentation.
