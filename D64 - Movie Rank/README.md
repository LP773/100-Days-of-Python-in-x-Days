# Movie Rank
In this project, I will be creating a Movie Ranking website utilizing
- Flask
- WTForms
- SQLite
- SQLAlchemy
and more

## Requirements
- [X] Be able to view movie list items 
- [X] Be able to edit a movie's rating
- [ ] Be able to delete movies from the database
- [ ] Be able to add new movies via the Add page
- [ ] Be able to sort and rank movies by rating
